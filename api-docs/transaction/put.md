# Muokkaa id:llä haettua myyntitapahtumaa

**URL**: `/transactions/:pk`

**Metodi**: `PUT`

**Vaadittu rooli**: Admin tai Myyjä

## Vastauksen paluukoodit

**Koodit**:

`200 OK`

`201 Created`

`204 No content`

## Esimerkkipyyntö:

**Polku**: `BASE_URL/transactions/1`

```json
{
  "appUser": { "userId": 2 }
}
```
