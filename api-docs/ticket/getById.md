# Näytä lippu ticketId:llä

**URL**: `/tickets/:pk`

**Metodi**: `GET`

**Vaadittu rooli**: Admin tai Myyjä tai Lipuntarkastaja

## Vastauksen paluukoodit

**Koodit**:

`200 OK`

`404 NOT FOUND`

> `"message": "Lippua ei löytynyt annetulla id:llä"`
