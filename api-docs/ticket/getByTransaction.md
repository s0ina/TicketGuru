# Hae kaikki myyntitapahtuman liput

**URL**: `/transactions/:pk/tickets` _:pk = transactionId_

**Metodi**: `GET`

**Vaadittu rooli**: Admin tai Myyj채 tai Lipuntarkastaja

**Koodit**:

`200 OK`

_Jos lippuja ei ole, API palauttaa tyhj채n taulukon koodilla 200 OK_
